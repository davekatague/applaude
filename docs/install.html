00 · Start here

Sixty seconds to your first win.

Confirm the foundation works before you go any further. If this works, everything else will too.

The test

Capture a thought by voice. See it on every device.

You don't need to install anything yet. You're just confirming Apple's iCloud sync is on.

  1. Open the Claude app on your iPhone. Tap the voice button.
  2. Say: "Add a reminder to my Captures list — I want to explore a voice-first second brain."
  3. Open the Reminders app on the same phone. The note should be in a list called Captures.
  4. Open Reminders on your Mac or iPad. It should appear within a minute, synced through iCloud.

That list — Captures 📥 — is your frictionless inbox. Everything goes in here first. Claude will help you process it.

If it didn't sync Settings → your name → iCloud → confirm Reminders is toggled on, on every device. About nine in ten "it isn't working" problems trace back to iCloud being off somewhere.

№ 01 · Mobile

Already on your phone.

The Claude app has three modes — voice, chat, and projects. All three can read and write your Reminders. Nothing to install.

Voice. Tap the microphone. Speak. Reminders just works — Claude on iOS uses Apple's own EventKit, the same thing Siri uses. No connector to turn on, no permissions to change.

Chat. Turn voice off and you can type, paste, and upload files. Switch voice back on and the conversation continues with the file still in context. Good for when you want to think something through slowly.

Projects. Inside a Claude.ai project, knowledge files you upload once are available across every voice and chat conversation in that project. This is where the system gets deep — your second brain's operating manual loaded as permanent context.

Try it now

  1. Open the Claude app. Tap the voice button to enter voice mode.
  2. Say: "Add a capture — I want to think about what I'd put in a second brain."
  3. Open Reminders. The thought is in your Captures 📥 list — your frictionless inbox.
  4. Come back to Claude and ask: "What's in my Captures right now?" Claude reads it back and can help you decide what to do with it.
№ 02 · Browser

Claude.ai. One click to connect.

Same Reminders, now in your browser. Five minutes. The connector is hosted by Anthropic — nothing to install on your computer.

This is the workhorse surface. Use it for longer thinking, planning, file uploads, and anything that benefits from a full screen. The Applaudé project lives here — it's your persistent second-brain interface, loaded with the operating manual Claude needs to structure your thoughts properly.

Turn on the Reminders connector

  1. Go to claude.ai and sign in.
  2. Open your profile menu → Connectors.
  3. Find Apple Reminders in the list and click Connect.
  4. Approve the permission prompt on your Mac. Allow access to Reminders.
  5. Open a new chat and type: "List my Reminders lists." You should see your lists come back.

Create the Applaudé project

The project is what gives Claude a persistent memory of how your system works. Without it, Claude treats every conversation as a blank slate.

  1. In Claude.ai, click Projects in the sidebar → New project. Name it Applaudé.
  2. In project knowledge, upload SKILL.md from the repo. This is the operating manual.
  3. Open project settings → paste the contents of bridges/claudeai.md as the project instructions.
  4. Start a new conversation inside the project. Claude will already know how your brain is structured.
  5. Say: "I just set up my Applaudé project. Can you check my lists and confirm you can see them?"
The four starter lists Create these in Reminders before your first session: Captures, Knowledge, Goals, Projects. That's all you need to start. Claude will refer to them as Captures 📥, Knowledge 🪎, Goals 🎯, and Projects 🚧 in conversation — the emoji is for warmth in chat; the plain name is what Reminders stores.
№ 03 · Desktop

Claude on your Mac.

The Mac app needs a small install — about fifteen minutes if you're comfortable in Terminal.

Before you start This path requires Terminal — the command-line app on your Mac. If that makes you nervous, stay on Claude.ai. You'll get most of the experience without it.
Option A

Stay on browser

Skip this section. Claude.ai gives you 90% of the experience with zero Terminal time.

Option B

Push through

Follow the steps below. You'll unlock Desktop and lay the groundwork for Claude Code too.

Claude Desktop on macOS doesn't have native Reminders access. It needs a small bridge — a community-built open-source server called mcp-server-apple-events that talks to EventKit, the same framework Apple's own apps use. One install covers Desktop and Claude Code both.

Install the bridge

  1. Download Claude Desktop from claude.ai/download and install it.
  2. Open Terminal (Applications → Utilities → Terminal).
  3. Open your Claude config folder: 
    open ~/Library/Application\ Support/Claude/
  4. Open claude_desktop_config.json in a text editor. If the file doesn't exist, create it.
  5. Paste this configuration (merge with anything already there): 
    {
  "mcpServers": {
    "apple-events": {
      "command": "npx",
      "args": ["-y", "mcp-server-apple-events"]
    }
  }
}
  6. Save the file. Quit Claude Desktop completely (⌘Q, not just close the window). Reopen it.
  7. The first time Claude reads Reminders, macOS will show a permission dialog. Click OK.
  8. Test: ask "List my Reminders lists." — you should see your four starter lists come back.

№ 04 · Claude Code

For developers.

Claude Code can read and write files on your Mac, run shell commands, and do work the other surfaces can't. If you don't write code, skip this — you don't need it.

Most of the setup is already done if you completed Desktop. Claude Code uses the same MCP bridge — you just point it at the Applaudé skill folder so Claude loads the operating manual automatically in every session.

Install

  1. Install Claude Code via npm: 
    npm install -g @anthropic-ai/claude-code
  2. Clone the Applaudé skill into your Claude skills folder: 
    git clone https://github.com/davekatague/applaude.git \
  ~/.claude/skills/applaude
  3. The Apple Events MCP bridge carries over from your Desktop install. No extra config needed.
  4. Start Claude Code in a new terminal. Say "list my Reminders lists" to verify. Then try: "capture this idea: [anything]."
05 · What to say

Your first five captures.

The most common question after setup: "what do I actually put in here?" Start with whatever is on your mind right now. Here are five shapes that work well.

Fleeting thought
"Capture this idea: what if pricing was based on outcomes, not hours?"
Goes to → Captures 📥. No structure needed. Just in. You'll decide what to do with it later.
Person you met
"I just met Alex Nguyen, a product manager at Stripe. Add her to People and note we talked about AI pricing models."
Goes to → People 🧱 (if you've created it) or Knowledge 🪎. Linked to context about when you met.
Lesson learned
"Log a lesson: I always underestimate API integration time. Double the estimate before committing."
Goes to → Knowledge 🪎 as a lesson.atomic. First-class in Applaudé — linked lessons keep you from repeating mistakes.
Goal
"New goal: finish the first draft of my essay by the end of June."
Goes to → Goals 🎯. With a clear success condition. Ask Claude later: "am I on track with my goals?"
Decision you made
"I decided to turn down that consulting offer. Reasons: scope kept expanding, rate didn't move. Log that."
Goes to → Knowledge 🪎 as a decision.atomic. Rationale preserved. Future-you can look back and understand why.
Something you read
"I just read that the most important skill in the next decade is knowing what to delegate to AI. Save that as a knowledge atom."
Goes to → Knowledge 🪎. Citable, retrievable, linkable to related ideas you'll add later.
The capture habit Don't try to structure while you capture. Rough goes into Captures 📥. On a weekly pass, you move the good stuff into Knowledge 🪎 or Projects 🚧. Trying to be perfect at capture kills the habit. Get it in. Structure later.
Seed your brain fast The repo includes a starter kit — five paste-ready atoms that give your system a structure from day one. Ask Claude: "Can you create my starter atoms from the starter-kit file?"

07 · What it feels like after a few weeks

Every idea you had. Still there.

This is the part that surprises people. Not the technology — just the feeling of it.

  • 🗣 Mid-drive, you remember something. You say it out loud. It's in your Captures list before you park.
  • 🔗 A month later, you mention a related topic. Claude surfaces the earlier thought, links the two, asks if they're connected. They are.
  • 🧠 You ask "what do I know about X?" You get back your own thinking — the decisions you made, the lessons you logged, the people who showed up in that context.
  • 📖 You review your goals. They're exactly as you left them. The reasoning is intact. Nothing got lost between conversations.
  • 🌱 It only gets richer. Every capture is a thread you can pull on later. The brain gets more useful the more you use it.

Where this came from

Standing on shoulders.

Applaudé wouldn't exist without two people who got there first.

Andrej Karpathy built a personal wiki that showed what it looks like to organise knowledge for reading. Click a link, follow a thought, learn something. Beautiful and clean.

Arscontexta took the same idea and asked: what if knowledge was organised for an agent to think with? Structured, linked, machine-readable.

Applaudé is the third step: knowledge organised so you can use it in real life — mid-drive, mid-call, mid-decision. By voice. On the devices you already own.
